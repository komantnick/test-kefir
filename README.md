## Тестовое задание
```
Реализовать на PHP-код, обрабатывающий запросы к страницам, соответствующим станциям метро.
Например, по адресу http://localhost/stancya/belyaevo должна открываться HTML-страница, в заголовке и текстах которой содержится название станции.
На странице должно быть 3 обязательных элемента:
1. Заголовок - "Ремонт одежды у метро [название метро]"
2. Герб города Москвы
3. Текст - "Порвалась любимая кофта? Прожгли джинсовую ткань? Сломался замок на куртке? Нужен недорогой мастер в столице? Вы на верном пути! Что бы ни приключилось – специалисты ближайшей к Вам мастерской возле метро [название метро] быстро справятся с неприятностью."
Компоновка, размещение, а также оформление (CSS-стили) блоков на усмотрение кандидата.
Результатом выполнения задания является список URL, соответствующий станциям метро, а также архив с файлами исходного кода.
Список станций метро можно получить из открытых источников. Например, https://metro.mwmoskva.ru/stancii-spisok/
```

## Структура проекта

```
./сss/                      - CSS-файлы проекта
./data/                     - внешние данные
./js/                       - JS-файлы проекта
./.htaccess                 - роутинг
./index.php                 - главная страница
./stancya.php               - страница с названием странции(режим доступа {site_name}/stancya/{station_name_in_translit})
```
